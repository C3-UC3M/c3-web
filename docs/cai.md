# Request Access

## How to request access to C3
1. Fill out [this form](https://aplicaciones.uc3m.es/formulario/c3_solicitud).

2. Once your request has been processed you will receive a budget by email. Reply to said email to accept or reject the budget.

3. Once you have accepted the budget you will receive credentials to access the cluster.

If you need more resources, please [contact us](mailto:c3-uc3m@uc3m.es).

## How to extend requested resources
If you need to extend the requested resources you must fill out a new form.
