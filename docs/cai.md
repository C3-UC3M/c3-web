# Request Access

## How to request access to C3
1. Register as a [UC3M CAI](https://www.uc3m.es/corefacilities/new-users) user. **Important for UC3M users**: you must specify the project code for invoicing. **Table of actual rates**:

| Concept                     | Rate (Euros)  |           |           |
|                             |               |           |           |
| **CAI User Type**           | **UC3M_A**    | **OPI_B** | **EXT_C** |
| CPU core per hour           | 0.008         | 0.02      | 0.03      |            
| GPU per hour                | 0.8           | 1.4       | 1.7       |      
| Storage (TB/month)          | 4             | 5         | 7         |

2. Once you are a CAI user you can request [CAI services](https://cai.uc3m.es/). See [this manual](https://cai.uc3m.es/media//manuales/MANUAL_CAI_LIMS_V_1.0.pdf) for help. In the description of the requested service please specify:
    1. CPU hours: minimum 1,000 hours, maximum 2,000,000 hours
    2. GPU hours: minimum 100 hours, maximum 2,000 hours
    3. Distributed storage (Lustre): minimum 1TB, maximum 4TB

3. Lastly, fill out [this form](https://aplicaciones.uc3m.es/formulario/c3_solicitud) for more specific information about resources and users.

If you need more resources, please [contact us](mailto:c3-uc3m@uc3m.es).


## How to extend requested resources
If you need to extend the requested resources you must apply for a new service. Follow **Steps 2 and 3**.
